# Arial

烂大街Sans-Serif字体，因为大部分计算机都带有该字体，因此这种字体常作为网站的第一字体或备用自体出现。

# Helvetica

烂大街Sans-Serif字体，Arial的前身，苹果设备默认英文字体。和Arial一样常作为网站的第一字体或备用自体出现。
